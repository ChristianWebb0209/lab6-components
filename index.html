Chat Component Approaches

  • Pure HTML and CSS Approach

    Markup with hardcoded messages. You can scroll and submit with the button, but no logic in place.

  • DOM Manipulation Approach

    Plain JavaScript to manipulate DOM and add the functionality. All bot logic gotten from eliza.js.

  • Progressive Enhancement Approach

    Web component that starts with semantic HTML and enhances it with JavaScript. Works even if JS fails to load.

  • Graceful Degradation Approach

    All CSS and most HTML encapsulated in shadow DOM, most of remaining functionality is the same as progressive enhancement approach
